Nivel Básico

  Variables y Tipos de Dato:

Variables:

  • Contenedores: Almacenan datos que pueden cambiar.
  • Tipos: `var`, `let`, `const` (cada uno con diferentes características de ámbito y mutabilidad).

Tipos de Datos:

Primitivos

Boolean: `true` o `false`.
Null: Valor nulo (ausencia intencional de valor).
Undefined: Valor indefinido (variable no inicializada).
Number: Números enteros o decimales.
String: Cadenas de texto.
Symbol: Valores únicos e inmutables.
BigInt: Números enteros grandes.


No Primitivos:

Object: Colecciones de datos (arrays, funciones, etc.).

Ejemplo:

javascript

let nombre = "Juan"; // String
const edad = 30; // Number
let esEstudiante = false; // Boolean

let persona = { // Object
nombre: "Ana",
edad: 25
};


En resumen:

  • Las variables almacenan datos.
  • Los tipos de datos definen el tipo de valor que puede contener una variable.
  • JavaScript tiene tipos primitivos (simples) y no primitivos (objetos).



  Declaracion de Variables

Declaración de Variables en JavaScript

En JavaScript, utilizamos las palabras clave `var`, `let` y `const` para declarar variables. Cada una tiene un ámbito y características diferentes:


var:

Ámbito: Funcional o global.
Reasignación: Se puede reasignar el valor.
Redeclaración: Se puede redeclarar en el mismo ámbito.
Hoisting: Las variables declaradas con `var` se "elevan" al principio del ámbito, lo que puede llevar a comportamientos inesperados.


let:

Ámbito: De bloque.
Reasignación: Se puede reasignar el valor.
Redeclaración: No se puede redeclarar en el mismo ámbito.
No hoisting: Las variables declaradas con `let` no se elevan.


const:

Ámbito: De bloque.
Reasignación: No se puede reasignar el valor.
Redeclaración: No se puede redeclarar en el mismo ámbito.
No hoisting: Las variables declaradas con `const` no se elevan.

Ejemplo:

javascript

// Declaración con var
var nombre = "Juan";

// Declaración con let
let edad = 30;

// Declaración con const
const PI = 3.14159;


Cuándo usar cada una:


var: Generalmente se evita debido a su comportamiento de hoisting y su ámbito funcional.
let: Se utiliza para variables que pueden cambiar de valor dentro de un bloque.
const: Se utiliza para constantes que no deben cambiar su valor.

Recomendaciones:
- Priorizar `let` y `const`: Son las formas más modernas y seguras de declarar variables.
- Evitar `var`: A menos que estés trabajando con código antiguo.
- Utilizar `const` por defecto: A menos que sepas que el valor va a cambiar.

En resumen:


`var`: Antiguo, ámbito funcional, se puede reasignar y redeclarar.
`let`: Moderno, ámbito de bloque, se puede reasignar pero no redeclarar.
`const`: Moderno, ámbito de bloque, no se puede reasignar ni redeclarar.
¡Recuerda! La elección de la palabra clave depende del contexto y de si necesitas que el valor de la variable sea mutable o inmutable.




  Números, Cadenas y Booleanos

Números (numbers)

Representan valores numéricos, tanto enteros como decimales.
Se utilizan para realizar operaciones matemáticas.
Ejemplo: `let edad = 30;`, `let pi = 3.14159;`

Cadenas (strings)

Secuencias de caracteres encerradas entre comillas simples o dobles.
Se utilizan para representar texto.
Ejemplo: `let nombre = "Juan";`, `let saludo = "Hola, mundo!";`

Booleanos (booleans)

Tienen solo dos valores posibles: `true` (verdadero) o `false` (falso).
Se utilizan para representar valores lógicos.
Ejemplo: `let esMayorDeEdad = true;`, `let estaLloviendo = false;`


Operaciones básicas:

  • Números: Suma (+), resta (-), multiplicación (*), división (/), módulo (%), incremento (++), decremento(--).
  • Cadenas: Concatenación (+), búsqueda (indexOf), reemplazo (replace),
    conversión a mayúsculas/minúsculas(toUpperCase(), toLowerCase()).
  • Booleanos: Comparaciones (==, ===, !=, !==), operadores lógicos (&&, ||, !).

Ejemplo de operaciones:

javascript


let numero1 = 10;
let numero2 = 5;
let resultadoSuma = numero1 + numero2; // resultadoSuma será 15

let nombre = "Ana";
let saludo = "Hola, " + nombre + "!"; // saludo será "Hola, Ana!"

let esMayor = 18 > 15; // esMayor será true


Conversión de tipos:

  • Puedes convertir un número a una cadena usando `String()`.
  • Puedes convertir una cadena a un número usando `Number()`.
  • Puedes convertir cualquier valor a un booleano usando `Boolean()`.

Ejemplo:

javascript

let numero = 42;
let cadena = String(numero); // cadena será "42"
let cadenaNumero = "10";
let numeroConvertido = Number(cadenaNumero); // numeroConvertido será 10


En resumen:


Losnúmerosse usan para cálculos.
Lascadenasse usan para texto.
Losbooleanosse usan para representar valores de verdad o falsedad.




  Operadores

  • Aritméticos
  • De Asignación
  • Comparación
  • Lógicos

Los operadores en JavaScript son símbolos especiales que realizan operaciones en uno o más valores (operandos). estos son los principales tipos de operadores:


Operadores Aritméticos:

Suma (+): Suma dos números.
Resta (-): Resta dos números.
Multiplicación (*): Multiplica dos números.
División (/): Divide dos números.
Módulo (%): Devuelve el resto de una división.
Incremento (++): Suma 1 al valor de una variable.
Decremento (--): Resta 1 al valor de una variable.
Potencia (**): Indica que un valor será elevado al otro.

Ejemplo:

javascript

let x = 10;
let y = 5;
let suma = x + y; // suma será 15
let resta = x - y; // resta será 5


Operadores de Asignación:

Igual (=): Asigna un valor a una variable.
Suma y asigna (+=): Suma un valor a una variable y asigna el resultado.
Resta y asigna (-=): Resta un valor a una variable y asigna el resultado.
Multiplicación y asigna (*=): Multiplica un valor a una variable y asigna el resultado.
División y asigna (/=): Divide un valor a una variable y asigna el resultado.

Ejemplo:

javascript

let a = 10;
a += 5; // a ahora es 15


Operadores de Comparación:

Igual a (==): Compara si dos valores son iguales.
Estrictamente igual a (===): Compara si dos valores son iguales en tipo y valor.
Diferente de (!=): Compara si dos valores son diferentes.
Estrictamente diferente de (!==): Compara si dos valores son diferentes en tipo o valor.
Mayor que (>): Compara si un valor es mayor que otro.
Menor que (<): Compara si un valor es menor que otro.
Mayor o igual que (>=): Compara si un valor es mayor o igual que otro.
Menor o igual que (<=): Compara si un valor es menor o igual que otro.

ejemplo:

javascript

let edad = 25;
let esMayorDeEdad = edad >= 18; // esMayorDeEdad será true


Operadores Lógicos:

Y (&&): Devuelve true si ambos operandos son true.
O (||): Devuelve true si al menos uno de los operandos es true.
No (!): Invierte el valor booleano.

Ejemplo:

javascript

let tienePermiso = true;
let tieneDinero = false;
let puedeComprar = tienePermiso && tieneDinero; // puedeComprar será false


Otros Operadores:

Operadores de cadena: Concatenación (+), búsqueda (indexOf), reemplazo (replace), etc.
Operadores bit a bit:Operan a nivel de bits (AND, OR, XOR, NOT).
Operador de tipoof: Devuelve el tipo de un valor.
Operador instanceof:Comprueba si un objeto es una instancia de un constructor.


Importante:
El orden de evaluación de los operadores sigue reglas de precedencia. Para asegurarte de que las operaciones se realicen en el orden correcto, puedes utilizar paréntesis.




  Condicionales

  • if
  • else if
  • else

Los condicionales en JavaScript nos permiten tomar decisiones en nuestro código, ejecutando diferentesbloques de código según se cumpla o no una determinada condición.


¿Qué son las condiciones?

Son expresiones que evalúan a `true` (verdadero) o `false` (falso).
Por ejemplo:

`5 > 3` (es verdadero)
`'hola' === 'mundo'` (es falso)


La estructura básica de un condicional es:

javascript

if (condición) {
// Código a ejecutar si la condición es verdadera
} else {
// Código a ejecutar si la condición es falsa
}

Ejemplo:

javascript

let edad = 18;

if (edad >= 18) {
console.log("Eres mayor de edad.");
} else {
console.log("Eres menor de edad.");
}


Condicionales anidados:

Puedes anidar múltiples condiciones para crear lógica más compleja:

javascript

let hora = 10;

if (hora < 12) {
console.log("Buenos días.");
} else if (hora < 18) {
console.log("Buenas tardes.");
} else
{ console.log("Buenas noches.");
}


Operadores lógicos:

Para combinar múltiples condiciones, utilizamos los operadores lógicos:
&& (y):Ambas condiciones deben ser verdaderas.
|| (o):Al menos una condición debe ser verdadera.
! (no):Niega una condición.

Ejemplo:

javascript


let tieneLicencia=true;
let tieneSeguro=false;

if (tieneLicencia && tieneSeguro) {
console.log("Puedes conducir.");
} else {
console.log("No puedes conducir.");
}


Operador ternario:

Es una forma más concisa de escribir un condicional simple:

javascript

let edad = 18;
let mensaje = (edad >= 18) ? "Eres mayor de edad" : "Eres menor de edad";
console.log(mensaje); // Imprimirá "Eres mayor de edad"


En Resumen:

Los condicionales nos permiten controlar el flujo de ejecución de nuestro código.
Utilizamos `if`, `else if` y `else` para crear diferentes ramas en nuestro código.
Las condiciones se evalúan a `true` o `false`.
Los operadores lógicos nos permiten combinar múltiples condiciones.
El operador ternario es una forma más corta de escribir condicionales simples.




  Bucles

  • for
  • do...while
  • while
  • for...of
  • for...in

Los bucles en JavaScript son estructuras que te permiten repetir un bloque de código varias vecesmientras se cumpla una determinada condición. Son fundamentales para automatizar tareas repetitivas.


Tipos de bucles:

while:

Ejecuta un bloque de código mientras una condición se evalúe como verdadera.

Sintaxis:
javascript

while (condición) {
// Código a repetir
}

Ejemplo:

javascript

let i = 0;
while (i < 5) { console.log(i); i++;
}


do...while:

Similar a`while`, pero garantiza que el bloque de código se ejecute al menos una vez, ya que la condición se verifica al final del bucle.

Sintaxis:
javascript

do {
// Código a repetir
} while (condición);


for:

Más versátil, permite inicializar una variable, establecer una condición y actualizarla en cada iteración.

Sintaxis:
javascript

for (inicialización; condición; actualización) {
// Código a repetir
}

Ejemplo:

javascript

for (let i = 0; i < 5; i++) {
console.log(i);
}


for...of

Permite iterar con muchos elementos de una variable de manera más sencilla

javascript

let animales = ["perro","gato","pez"]
for (let animal of animales) {
console.log(animal) // imprime cada uno de los elementos del array
}

Más explicito

let animales = ["perro","gato","pez"]
// es la declaración del array

for (let animal of animales) {
// el nombre de la variable [let animal]
// indica como se llamarían los elementos ["perro","gato","pez"]
// dentro del array [animales]
// dentro de las llaves {} lo que se haría con esa declaracion de elementos y array

En el ejemplo:
console.log(animal) }
// se imprimen los elementos contenidos en el array, ya que "animal" fue el nombre que recibió el conjunto de elementos contenidos en el array


for...in

Permite iterar con todas las propiedades de un elemento en dado caso que no las sepas

let user = {
  id: 1,
  nombre: Juan,
  age: 20,
}; // tenemos acá una variable con valores asignado a las propiedades

javascript

for (let prop in user) {
  console.log(prop, user[prop]) //paréntesis cuadrado para acceder a la propiedad
} //imprime la propiedad con sus valores

Más explicito

let personas = ["carlos", "juan", "alberto"];
// este codigo declara el array que contiene nombres

for (let indice in personas) {
// declara la variable "indice" que en for in, sólo busca el indice y no los elementos que contiene el array "personas".
// para personas busca el indice de cada elemento que contiene, sólo el indice.

console.log(indice, personas[indice])
// el console.log busca el "indice" - "0,1,2" que existe.
// luego se le pide que dentro de la variable "personas" busque los elementos del indice
// "[]" los signos le indican a algoritmo que deben ingresar y buscar los elemento contenidos en la variable personas.


¿Cuándo usar cada uno?

while:Cuando no sabes de antemano cuántas veces se ejecutará el bucle, pero tienes una condición clara para detenerlo.

do...while:Cuando necesitas que el código se ejecute al menos una vez, incluso si la condición inicial es falsa.

for:Cuando tienes un contador y sabes exactamente cuántas veces quieres que se repita el bucle.


Otras consideraciones:

break:Interrumpe la ejecución del bucle inmediatamente.

continue:Salta a la siguiente iteración del bucle.

Etiquetas:Puedes etiquetar bucles para controlar `break` y `continue` en bucles anidados.


Switch

Similar a if con else if, se puede utilizar dependiendo de la preferencia.

javascript

let accion = "listar";

switch (accion) {
  case "listar":
    console.log("Acción de listar");
    break;
  case "guardar":
    console.log("Acción de guardar");
    break;
  default:
    console.log("Acción no reconocida");
}

// mostraría en consola el console.log correspondiente al case listar.


Ejemplo con break y continue:

javascript

for (let i = 0; i < 10; i++) {
  if (i === 5) {
    break; // Detener el bucle cuando i sea 5
 }
  if (i % 2 === 0) {
    continue; // Saltar los números pares
 }
  console.log(i); // Imprime solo los números impares hasta 4
}


En resumen:

Los bucles son herramientas poderosas en JavaScript que te permiten automatizar tareas repetitivas. La elección del tipo de bucle depende de la lógica de tu programa y de la cantidad de veces que necesites repetir una acción.




  Funciones

definición y algunos ejemplos

Las funciones en JavaScript son como pequeños bloques de código reutilizables que realizan tareas específicas.
Imagina que son como recetas: tienes una serie de pasos para hacer un pastel, y cada vez que quieres un pastel, sigues esa misma receta. En programación, las funciones son como esas recetas.


¿Qué es una función en JavaScript?

Bloque de código:Es un conjunto de instrucciones que se ejecutan cuando se llama a la función.

Reutilizable:Puedes definir una función una vez y llamarla muchas veces desde diferentes partes de tu código.

Parámetros:Pueden recibir datos (parámetros) para personalizar su comportamiento.

Retorno:Pueden devolver un resultado (valor de retorno).


Sintaxis básica:

javascript

function nombreDeLaFuncion(parametro1, parametro2, ...) {
  // Código que realiza la tarea
  return valorDeRetorno; // Opcional
}

Ejemplo:

javascript

function saludar(nombre) {
  console.log("Hola, " + nombre + "!");
}

saludar("Juan"); // Imprime: Hola, Juan!


¿Para qué sirven las funciones?

Organizar el código: Dividir el código en funciones más pequeñas hace que sea más fácil de leer y mantener.

Reutilización:Evita repetir el mismo código varias veces.

Abstracción:Puedes ocultar la complejidad de ciertas operaciones dentro de una función.

Modularidad:Permite crear módulos de código independientes.


Tipos de funciones:

Declarativas:Se declaran con la palabra clave`function`.

Expresivas:Se asignan a una variable.

Flecha:Una sintaxis más concisa para funciones.


Ejemplo de función flecha:

javascript

const sumar = (a, b) => {
  return a + b;
};


¿Por qué son importantes las funciones?

Las funciones son uno de los pilares fundamentales de la programación en JavaScript. Te permiten escribir código más limpio, eficiente y fácil de mantener. Al comprender cómo funcionan las funciones, podrás crear aplicacionesJavaScript más robustas y escalables.




  Llamar a la función y parametros

En JavaScript, llamar a una función es como darle la orden de ejecutar el código que contiene. Es como decir: "Oye, función, haz lo que tienes que hacer".


¿Cómo se llama a una función?

Simplemente escribes el nombre de la función seguido de paréntesis, así:

javascript

nombreDeLaFuncion();

Ejemplo:

Imagina que tienes una función llamada `saludar` que muestra un mensaje en la consola. Para llamarla, escribirías:

javascript

function saludar() {
  console.log("¡Hola!");
}

saludar(); // Esto mostrará "¡Hola!" en la consola


¿Qué pasa cuando llamas a una función?

1.El programa llega a la línea donde está la llamada a la función.

2.Se detiene la ejecución del código actual y se "salta" al código dentro de la función.

3.Se ejecuta el código de la función, paso a paso.

4.Una vez que se termina de ejecutar el código de la función, el programa vuelve al punto donde lo dejó y continúa ejecutando el resto del código.


¿Qué son los argumentos?

A veces, las funciones necesitan información adicional para realizar su tarea. Esta información se proporciona a través de argumentos . Los argumentos se colocan dentro de los paréntesis al llamar a la función.

Ejemplo con argumentos:

javascript

function saludar(nombre) {
  console.log("¡Hola, " + nombre + "!");
}

saludar("Ana"); // Imprimirá: ¡Hola, Ana!


En resumen:

Llamar a una función es como activar una acción en tu programa. Al llamar a una función, le estás diciendo al programa que ejecute el código que se encuentra dentro de esa función. Los argumentos son como datos que le proporcionas a la función para que pueda realizar su tarea de manera más personalizada.




Retorno de Valores (Return)

El retorno de valores en JavaScript es una característica fundamental de las funciones que te permite obtener un resultado específico después de ejecutar un bloque de código.


¿Qué significa retornar un valor?

Cuando una función ejecuta su código y llega a una instrucción`return`, el valor que sigue a esa palabra clave se devuelve al lugar donde se llamó a la función. Es como si la función entregara un resultado a quien la llamó.

Sintaxis:

javascript

function miFuncion(parametro1, parametro2) {
  // Código de la función
return valorADevolver;
}

Ejemplo:

javascript

function sumar(a, b) {
  let resultado = a + b;
return resultado;
}

let suma = sumar(3, 5); // suma tendrá el valor 8

En este ejemplo:

La función`sumar`toma dos números como parámetros.

Dentro de la función, se suman los dos números y se almacena el resultado en la variable`resultado`.

La instrucción`return resultado`devuelve el valor de`resultado`al lugar donde se llamó a la función.

El valor devuelto se asigna a la variable`suma`.


¿Para qué sirve el retorno de valores?

Realizar cálculos:Las funciones pueden realizar cálculos complejos y devolver el resultado final.

Manipular datos:Pueden transformar datos de entrada en un formato diferente.

Tomar decisiones:El valor devuelto puede ser utilizado en condicionales para tomar decisiones.

Crear funciones más versátiles: Al devolver valores, las funciones pueden ser utilizadas en una variedad de contextos.


Ejemplo con condicional:

javascript

function esMayorDeEdad(edad) {
  return edad >= 18;
}

let puedeVotar = esMayorDeEdad(20); // puedeVotar será true

if (puedeVotar) {
  console.log("Puedes votar");
}

En resumen:

El retorno de valores en JavaScript permite a las funciones entregar un resultado específico después de su ejecución. Este resultado puede ser utilizado en otras partes del código para realizar cálculos, tomar decisiones o simplemente almacenar un valor. Es una herramienta fundamental para crear funciones más poderosas y flexibles.
